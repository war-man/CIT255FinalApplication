Title: Weather To Plant
Description: NMC CIT 255 Object Oriented Programming course capstone project: a C# .NET application that pulls weather data from the Wunderground API and displays planting suggestions for educational purposes
Application Type: Console (with intention to transition to WinForms)
Persistence Type: XML
Persistence Location: Web API
Date Created: 12/2/2017
Last Modified: 12/7/2017
Instructions For Use: Pull weather from the Wunderground API and see the 10-day forecast with "rainy" days including all days with a percentage chance of precipitation higher than 50%.  You can see suggested planting days by choosing "Auto Fill", or select your own planting days by choosing "Custom".  You can also print your schedule.
